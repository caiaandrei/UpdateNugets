# Update NuGets

Update NuGets is a Windows app that simplifies the process of updating different NuGets from different projects.

![UpdateNuGetsApp](https://user-images.githubusercontent.com/66559398/113614637-6ecc3880-965b-11eb-83d6-6c703b964d59.png)

## Features

- Shows all NuGets used in an app and the project files where these are used
- Update the version of a NuGet in all of the projects where it's being used
- Show basic information about a NuGet (e.g. Name, Available Versions, Dependencies)
- Generate an Excel which contains the previous version and the new version of a NuGet
